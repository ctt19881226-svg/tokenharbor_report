# Executive Summary Spec v1

Purpose:

Generate a concise founder-focused summary.

File:

executive_summary.txt

Requirements:

- 3 to 5 bullet points
    
- Maximum 120 words
    
- Focus only on meaningful changes
    
- Avoid repeating every metric
    
- Highlight anomalies and trends
    

Priority Order:

1. User Growth
    
2. Activation
    
3. Revenue
    
4. Usage
    
5. Top Models
    
6. Referral Growth
    
7. Website Traffic
    

Good Example:

• 6 new users registered yesterday, bringing total users to 127.

• Activation remained healthy with 4 new activated users and a 33.1% activation rate.

• Revenue increased by $20 from one new paying user.

• GPT-5.5 and Claude Sonnet 4.6 accounted for 73% of all requests.

• Product Hunt generated 21% of website traffic and remained the second-largest traffic source.

Bad Example:

• Total Users: 127  
• New Users: 6  
• Verified Users: 81  
• Revenue: 20

This is metric repetition and provides no insight.
